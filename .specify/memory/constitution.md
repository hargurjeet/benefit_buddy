# BenefitBuddy Project Constitution

* **Core Stack**: Python, Google ADK 2.0, gemini-2.0-flash.
* **Data Layer**: Local vector store (Chroma/FAISS) for Kaggle dataset + Serper API for live search.
* **Hard Constraints**: All external sources must strictly follow a `.gov.in` or `.nic.in` allowlist. No agent may fabricate scheme names.
